# Employee Attrition and Performance Analysis

This project explores patterns and correlations in IBM's HR dataset to understand why employees leave and how to improve retention and performance.

## 📁 Project Structure
- `data/`: Dataset file
- `notebook.ipynb`: Code and visualizations
- `visuals/`: Saved plots and charts
- `presentation/`: Final slides or video
## 📊 Job Role vs. Attrition & Overtime – HR Insights Dashboard

This interactive Tableau dashboard visualizes the relationship between employee attrition and overtime percentage across different job roles.

### Key Features:
- **Attrition Rate by Job Role** – Highlights roles with high employee turnover.
- **Overtime % by Job Role** – Identifies roles with the most workload stress.
- **Interactive Filters** – Explore specific roles or compare patterns easily.

### 🎯 Objective:
To identify trends that may explain higher attrition in specific job roles, particularly in relation to overtime.
